  • 사용자 자가 측정 계산기 UI 퍼블리싱: 단순 정보 나열 방식에서 벗어나 유저가 직접 수치를 입력해 결과를 확인하는 계산기 UI를 구현했습니다. 이를 통해 유저의 자발적 정보 획득을 돕고 운영 측면의 단순 문의(CS)를 줄이는 효과를 거두었습니다.
  • 비대면 서류 제출 시스템 구축 지원: 기존 오프라인 방식(팩스)을 대체하는 온라인 파일 업로드 시스템의 사용자 및 관리자 UI 퍼블리싱을 담당했습니다. 실무 프로세스의 디지털 전환 과정을 경험하며 인터랙션 설계의 중요성을 익혔습니다.
  • 멀티 디바이스 웹 호환성 검수 및 수정: 다양한 모바일 기기와 브라우저 환경에서의 레이아웃 정합성을 검수했습니다. 실제 기기 테스트를 통해 발견된 미세한 오차를 1px 단위로 수정하며 크로스 브라우징 최적화 역량을 길렀습니다.
